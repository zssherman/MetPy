relative_humidity_from_dewpoint¶

metpy.calc.relative_humidity_from_dewpoint(temperature, dewpt)[source]¶

Calculate the relative humidity.

Uses temperature and dewpoint in celsius to calculate relative humidity using the ratio of vapor pressure to saturation vapor pressures.

Parameters:
  • temperature (pint.Quantity) – The temperature
  • point (dew) – The dew point temperature
Returns:

pint.Quantity – The relative humidity

See also

saturation_vapor_pressure()
